# 📱 Prova-Leonardo

Projeto desenvolvido no Android Studio como parte de uma atividade acadêmica.
O aplicativo tem como objetivo demonstrar conceitos fundamentais de desenvolvimento Android, como exibição de listas, adaptação de dados e estruturação de código.

---

## 🚀 Funcionalidades

* 📋 Exibição de lista de produtos
* 🔄 Uso de Adapter para renderização dinâmica (`ProductAdapter`)
* 🧱 Estrutura organizada em pacotes
* 📱 Interface simples e funcional

---

## 🛠️ Tecnologias utilizadas

* Java / Kotlin (dependendo do seu projeto)
* Android Studio
* RecyclerView / ListView
* Gradle

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/AlexandreGSSantos/Prova-Leonardo.git
```

2. Abra no Android Studio

3. Aguarde o Gradle sincronizar

4. Execute o projeto em:

   * Emulador Android
     ou
   * Dispositivo físico
